# Contributing

1. Fork the repository
2. Create a branch: `git checkout -b feat/your-feature`
3. Install dependencies: `pnpm install`
4. Make changes, run `pnpm test` and `pnpm lint`
5. Commit using [Conventional Commits](https://www.conventionalcommits.org/): `type(scope): description`
6. Open a pull request

**Types:** `feat`, `fix`, `docs`, `refactor`, `test`, `chore`, `ci`

See [docs/CODE_OF_CONDUCT.md](docs/CODE_OF_CONDUCT.md) for community guidelines.

Questions? Open a [GitHub Discussion](../../discussions) or file an issue.
